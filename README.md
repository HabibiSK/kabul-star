## 🎨 Design-System: Midnight Gilded

### Colors

| Token | Hex | Usage |
|-------|-----|-------|
| `surface` | `#0c1324` | Main background |
| `primary` | `#f2ca50` | Gold highlights |
| `primary-container` | `#d4af37` | **Heritage Gold** |
| `on-surface` | `#dbe2fa` | Primary text |
| `on-surface-variant` | `#d0c5af` | Secondary text |

### Typography

| Token | Font | Size | Usage |
|-------|------|------|-------|
| `display-lg` | Noto Serif | 64px | Hero headlines |
| `headline-lg` | Noto Serif | 48px | Section titles |
| `body-lg` | Manrope | 18px | Lead text |
| `label-caps` | Manrope | 12px | Labels, nav |
